# Компактный справочник шагов Vanessa Automation

## Наиболее используемые шаги по категориям


### UI.Окна

```gherkin
И я закрываю все окна клиентского приложения
```
*Закрывает все окна текущего клиента тестирования.*

```gherkin
И я закрываю все окна клиентского приложения кроме "Нужное окно"
```
*Закрывает все окна текущего TestClient, кроме нужного окна.*

```gherkin
Тогда открылось окно "Заголовок окна"
```
*Проверяет, что на экране есть активное окно с нужым заголовком. Возможно передавать сразу несколько заголовков окон через таблицу Gherkin.*

```gherkin
Тогда открылась форма "Справочник.Справочник1.Форма.ФормаЭлемента"
```
*Проверяет, что на экране есть активное окно с нужной формой. Возможно передавать сразу несколько заголовков форм через таблицу Gherkin.*

```gherkin
Тогда открылась форма с именем 'Справочник.Справочник1.Форма.ФормаЭлемента'
```
*Проверяет, что на экране есть активное окно с нужной формой. Возможно передавать сразу несколько имен форм через таблицу Gherkin.*

```gherkin
И я активизирую окно "Заголовок окна"
```
*Активизирует окно с нужным заголовком*

```gherkin
И я активизирую форму "Заголовок формы"
```
*Активизирует окно, в котором будет найдена форма с нужным заголовком.*

```gherkin
И я активизирую форму с именем "ИмяФормы"
```
*Активизирует окно, в котором будет найдена форма с нужным именем.*

```gherkin
И я жду закрытия окна "Заголовок окна" в течение 20 секунд
```
*Ожидает, что окно исчезнет в течение указанного числа секунд. Поиск окна происходит среди всех открытых окон. Поэтому шаг будет ждать пока закроются все окна, удовлетворяющие переданному параметру "Заголовок окна".*

```gherkin
И я жду открытия окна "Заголовок окна" в течение 20 секунд
```
*Ожидает, что окно появится в течение указанного числа секунд. Окно не обязательно должно быть активным окном. Поиск производится среди всех окон.*

```gherkin
И я жду, что активным окном станет "Заголовок окна" в течение 10 секунд
```
*Ожидает, что указанное окно станет активным окном в течение указанного времени.*

```gherkin
И я жду открытия окна отличного от "Заголовок окна" в течение 20 секунд
```
*Ожидает что откроется другое окно, отличное от переданного в течение указанного числа секунд. Используется, чтобы определить смену окон.*

```gherkin
И я закрываю окно "Заголовок окна"
```
*Закрывает окно с нужным заголовком*

```gherkin
И я закрываю текущее окно
```
*Закрывает активное окно*

```gherkin
И я сохраняю навигационную ссылку текущего окна в переменную "ИмяПеременной"
```
*Запоминает навигационную ссылку текущего окна, если оно находится в панели открытых окон.*

```gherkin
И я запоминаю заголовок текущего окна как "ЗаголовокОкна"
```
*Сохранение значения заголовка текущего окна в переменную.*

```gherkin
И я запоминаю количество окон в переменную "ИмяПеременной"
```
*Сохраняет в переменную количество открытых окон в текущем сеансе клиента тестирования.*

```gherkin
Дано Открылось окно предупреждения с заголовком "Заголовок окна" и текстом "Текст"
```
*Шаг проверяет, что сейчас на экране появилось окно предупреждения.*

```gherkin
Дано В активном окне открылась форма с заголовком "Заголовок формы"
```
*Шаг проверяет, что в активном окне есть нужная форма.*

```gherkin
Если появилось окно с заголовком "Заголовок окна" Тогда
```
*Условие. Проверяет, что среди всех окон есть окно с нужным заголовком*

```gherkin
Если текущее окно имеет заголовок "Заголовок окна" Тогда
```
*Условие. Проверяет, что в данный момент активным окном является окно с нужным заголовком*

```gherkin
Если текущее окно "не равно" ожидаемому окну "Заголовок окна" Тогда
```
*Условие. Проверяет что текущее окно равно/неравно нужному окну.*

```gherkin
Если появилось окно с заголовком из таблицы  Тогда
```
*Условие. Проверяет, что есть окно с заголовком из таблицы*

```gherkin
Если это начальная страница Тогда
```
*Условие. Проверяет, то текущее окно это начальная страница*

```gherkin
Если это не начальная страница Тогда
```
*Условие. Проверяет, то текущее окно это не начальная страница*

```gherkin
Если появилось окно с заголовком "Заголовок окна" в течение 10 секунд Тогда
```
*Условие. Проверяет, что на экране появилось (не обязательно является активным) окно с нужным заголовком в течение нужного числа секунд*
